# QtGPS
This is simple Qt/cpp based Andriod APK to view Latitude , Longitude and speed and Accuracy on Android device.

##Dependencies 
1. Android SDK >= 20.0
2. NDK >= 26.1.xxxx
3. JDK
4. Qt >= 6.5

## Qmake build
It's compiled and tested with Qt6.9.3 and Android_arm64_v8a. COnfigure Android in Qt Creator and build with Android_arm64_v8a tool.
